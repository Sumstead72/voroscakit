# VoroscakIT Brand Palette

## Backgrounds

| Token        | Hex       | Usage                        |
|--------------|-----------|------------------------------|
| Primary      | `#0b1a2e` | Main background              |
| Secondary    | `#0f2137` | Alternate sections           |
| Card         | `#132a42` | Card surfaces                |
| Card Hover   | `#183454` | Card hover state             |

## Accent

| Token       | Value                        | Usage                  |
|-------------|------------------------------|------------------------|
| Accent      | `#4ecdc4`                    | Primary accent (teal)  |
| Accent Dim  | `rgba(78, 205, 196, 0.15)`  | Subtle backgrounds     |
| Accent Glow | `rgba(78, 205, 196, 0.3)`   | Glow / shadow effects  |

## Text

| Token     | Hex       | Usage                          |
|-----------|-----------|--------------------------------|
| Primary   | `#ffffff` | Headings, body text            |
| Secondary | `#8fa4be` | Descriptions, secondary copy   |
| Muted     | `#5a7088` | Labels, fine print             |

## Borders

| Token        | Value                        | Usage                      |
|--------------|------------------------------|----------------------------|
| Border       | `rgba(255, 255, 255, 0.06)` | Card / section borders     |
| Border Accent| `rgba(78, 205, 196, 0.2)`   | Hover borders, accent lines|

## Logo Usage

- **"Voroscak"** — `#ffffff` (white)
- **"IT"** — `#4ecdc4` (teal) with `text-shadow: 0 0 20px rgba(78, 205, 196, 0.3)`

## Typography

- **Display / Mono:** JetBrains Mono (logo, labels, stats)
- **Body:** Outfit (headings, paragraphs, buttons)
